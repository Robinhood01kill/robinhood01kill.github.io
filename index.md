## 🔧 Frequently asked questions

**Do the agents have access to my private keys?**
The software stores your sensitive credentials in an encrypted file on your local machine. It does not send these keys to any external server.

**Can I run the software while my computer is locked?**
Yes, the agents continue to operate as long as the application remains open and the computer maintains an internet connection.

**What happens if my internet disconnects?**
The agents enter a safe mode and pause all active trades until you restore the connection.

**Does the software require a subscription?**
The current version functions as a standalone tool. No recurring fees apply.

**How do I backup my configuration?**
Navigate to the "Settings" menu inside the application and select "Export configuration." This saves a file that you can restore later if you reinstall the software.
